# DeckForge — Plan

## What does this project do?

`lewislulu/html-ppt-skill` (⭐1122) is a browser-based AI presentation builder: paste a text prompt, get a styled multi-slide HTML deck generated by Claude. 24 themes, 31 layouts, 20+ animations. Exports as a single self-contained HTML file you can open or share anywhere.

## Where does it fit?

**Personal tooling for Daniel.** He's Head of Product AI/Automation/Integrations — regularly pitches ideas, explains architectures, demos agentic systems. A fast AI slide generator saves real time.

## Scoped MVP

- Text prompt → Claude generates structured slide JSON → rendered deck
- 4 polished themes (dark/midnight/glass/light) matching our aesthetic
- 7 layouts: title, bullets, content, two-col, quote, code, closing
- Household-specific seed prompts (Agent Household Overview, ClawDash Architecture, Podda Pitch, etc.)
- Fullscreen present mode (keyboard nav)
- Export as single self-contained HTML file — the killer feature
- API key stored in localStorage (user provides own Anthropic key)

## Real data / household context

- Seed prompts pre-loaded for: Agent Household Overview, ClawDash Architecture, Podda App Pitch, VPS Infrastructure, Weekly Product Demo, ClawBridge Protocol
- These give immediate value without needing custom input

## Build tasks

1. Scaffold Vite + React + TypeScript + Tailwind CSS v4
2. Types: Slide, Deck, Theme, Layout
3. Claude API client (direct fetch with `anthropic-dangerous-direct-browser-access: true`)
4. Slide renderer (7 layouts × 4 themes via CSS variables)
5. Deck preview panel (slide rail + main view)
6. Present mode (fullscreen + keyboard nav)
7. HTML export (single self-contained file)
8. Prompt panel with household seed prompts
9. API key setup modal
10. Build + deploy to mvp.trollefsen.com/2026-04-18-deckforge/
